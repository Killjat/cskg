# 网站信息收集工具 - 项目总结

## 项目概述

这是一个功能全面的网站信息收集工具，专门用于提取和分析网站的基础信息、备案信息、技术栈信息等关键数据。

## 核心功能

### 1. 基础信息收集
- **网站标题**: 提取网页title标签内容
- **描述信息**: 提取meta description和keywords
- **语言和字符集**: 检测网站语言和编码
- **作者信息**: 提取meta author信息

### 2. 图标信息提取
- **Favicon**: 网站图标链接
- **Apple Touch Icon**: iOS设备图标
- **多尺寸图标**: 收集各种尺寸的图标信息
- **图标类型**: 识别图标格式和用途

### 3. 备案信息识别 ⭐
- **ICP备案号**: 智能识别工信部ICP备案信息
- **网安备案号**: 提取公安部网安备案号
- **备案组织**: 识别备案主体组织名称
- **多模式匹配**: 支持多种备案号格式识别

### 4. 文件下载链接发现
- **文件类型识别**: 支持PDF、DOC、图片、视频等多种格式
- **下载链接提取**: 自动发现页面中的文件下载链接
- **文件信息**: 提取文件名、类型、上下文信息
- **批量发现**: 深度爬取发现更多文件资源

### 5. 页脚信息解析
- **版权信息**: 提取copyright声明
- **联系方式**: 智能识别邮箱、电话、QQ等联系信息
- **页脚链接**: 收集页脚导航链接
- **社交媒体**: 识别微博、微信、QQ等社交平台链接

### 6. 技术栈检测
- **Web服务器**: 识别Nginx、Apache等服务器
- **编程语言**: 检测PHP、Python等后端技术
- **CMS系统**: 识别WordPress、Drupal、Discuz等
- **前端框架**: 检测jQuery、Bootstrap、Vue等
- **分析工具**: 发现Google Analytics、百度统计等

## 技术架构

### 核心模块
```
web-info-collector/
├── main.go           # 主程序和CLI接口
├── types.go          # 数据结构定义
├── collector.go      # 信息收集核心逻辑
├── batch.go          # 批量处理和导出
└── examples/         # 示例文件和测试数据
```

### 关键特性
- **智能解析**: 基于正则表达式和DOM解析的智能信息提取
- **并发处理**: 支持多线程批量网站分析
- **深度爬取**: 可配置的爬取深度和页面数量限制
- **多格式输出**: JSON、CSV、HTML三种输出格式
- **错误处理**: 完善的错误处理和重试机制

## 使用示例

### 基本信息收集
```bash
./web-info-collector -u https://www.baidu.com
```

### 深度爬取
```bash
./web-info-collector -u https://example.com -d 3 --max-pages 50
```

### 批量分析
```bash
./web-info-collector -f urls.txt -c 10 -o results.json
```

### HTML报告生成
```bash
./web-info-collector -f urls.txt --format html -o report.html
```

- **竞争对手调研**: 分析竞品网站技术架构
- **行业技术趋势**: 统计行业内技术使用情况
- **联系方式收集**: 获取目标企业联系信息

### 3. 安全研究
- **信息收集**: 安全测试前的信息收集阶段
- **资产发现**: 发现目标组织的相关资产
- **技术指纹**: 识别目标系统的技术特征

### 4. 市场调研
- **行业分析**: 批量分析特定行业网站特征
- **备案统计**: 统计特定地区或行业的备案情况
- **技术采用率**: 分析新技术在市场中的采用情况

## 性能特点

- **高效爬取**: 平均每个网站2-5秒完成信息收集
- **并发处理**: 支持最多50个并发连接
- **内存优化**: 流式处理大批量数据
- **智能限速**: 可配置请求间隔避免被封

## 技术亮点

### 1. 智能备案信息识别
- **多模式匹配**: 支持各种备案号格式
- **上下文分析**: 结合页面上下文提高识别准确率
- **省份识别**: 自动识别备案省份信息

### 2. 文件链接发现
- **类型识别**: 支持30+种文件格式
- **上下文提取**: 记录文件链接的页面上下文
- **批量发现**: 深度爬取发现更多文件资源

### 3. 联系信息提取
- **多格式支持**: 识别各种电话、邮箱格式
- **社交媒体**: 自动识别微博、微信等社交平台
- **智能过滤**: 过滤无效和重复的联系信息

### 4. 技术栈检测
- **多维度检测**: 从HTTP头、HTML内容、JavaScript等多个维度检测
- **版本识别**: 尽可能识别技术组件的版本信息
- **CMS指纹**: 基于特征文件和路径识别CMS系统

## 扩展性
